Linguagem CSS

O Cascading Style Sheets (CSS) é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML.
O CSS descreve como os elementos HTML devem ser exibidos na tela, no papel ou em outras mídias.

Propriedades CSS

As propriedades CSS são usadas para definir o estilo de um elemento HTML.
Elas incluem propriedades como cor, tamanho, margens, preenchimento e muitas outras.

  • color: define a cor do texto
  • font-size: define o tamanho da fonte
  • margin: define as margens do elemento
  • padding: define o preenchimento interno do elemento

Box Model

O Box Model é um conceito fundamental do CSS que define a estrutura de cada elemento na página.
Ele consiste em margem, borda, preenchimento e conteúdo.

Conclusão

O CSS é essencial para o design e a apresentação de páginas da web. Com os seletores e propriedades corretos,
é possível criar layouts atraentes e responsivos. Além disso, o CSS trabalha em conjunto com o HTML e o JavaScript para fornecer uma experiência completa na web.
